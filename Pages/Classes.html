1. RnaSequence Class

We created this class to have a datatype definition for RnaSequence. We implemented methods to get the length of the RnaSequence and to get the pair at a particular index.

2. RnaFolding Class

This class was created to have a better understanding on which of the bases are being paired and which ones are being left out according to our optimal algorithm. We implemented the method tracebackSecondaryStructure in this class. While filling the dp table , if the index is not taken for pairing, we store -1 as the splitpoint in the traceback table, in the other case if pairing happens , we choose a t and store it as the splitpoint in the traceback table. We use a stack to check iteratively if there was any split for 0,n-1 initially, once we find the split we break into the left and right side and push it into the stack, the algorithm runs iteratively in this manner.
