2. canPair()

3. solve()

  • The algorithm uses dynamic programming to efficiently solve the problem by breaking it down into overlapping subproblems.
  • It iteratively builds up solutions for subsequences of increasing lengths, optimizing the partitioning of the RNA sequence to maximize the number of base pairs formed.
  • The OPT matrix stores optimal scores, and the traceback matrix facilitates backtracking to reconstruct the optimal solution once the DP table is filled.

   

4. getTraceback()

This function returns the traced back string.

5. tracebackSecondaryStructure()

  • The traceback mechanism reconstructs the secondary structure of an RNA sequence based on the traceback matrix generated by a dynamic programming algorithm.
  • It uses a stack-based approach to traverse and process segments of the RNA sequence, updating a structure string (structure) to denote base pairing and collecting paired indices in a pairs vector.
  • The reconstructed secondary structure (base pairs) is returned as a vector of pairs representing the paired indices.
